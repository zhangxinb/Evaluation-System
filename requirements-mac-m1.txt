# Installation instructions for Apple Silicon:
# 
# 1. Ensure you're using Python 3.9+ (arm64 version, not x86_64)
#    Check with: python3 -c "import platform; print(platform.machine())"
#    Should output: arm64
# 
# 2. Install Xcode Command Line Tools:
#    xcode-select --install
# 
# 3. Create virtual environment:
#    python3 -m venv venv
#    source venv/bin/activate
# 
# 4. Upgrade pip:
#    pip install --upgrade pip setuptools wheel
# 
# 5. Install dependencies:
#    pip install -r requirements-mac-m1.txt
# 
# 6. If you encounter issues with specific packages:
#    - For OpenCV: brew install opencv
#    - For TensorFlow: It should work natively on M1/M2
#    - For PyTorch: Use MPS backend for GPU acceleration
# 
# 7. To use Metal Performance Shaders (MPS) for GPU acceleration:
#    In your code, use: device = torch.device("mps") if torch.backends.mps.is_available() else torch.device("cpu")
# 
# 8. Known issues:
#    - Some packages may require compilation from source
#    - If segment-anything fails, you can comment it out (optional feature)
#    - DeepFace model downloads happen on first run

# Performance tips for M1/M2:
# - MPS (Metal) provides GPU acceleration for PyTorch
# - TensorFlow has native Metal support
# - Use unified memory architecture advantage
# - Set appropriate thread counts: os.environ['OMP_NUM_THREADS'] = '8'
